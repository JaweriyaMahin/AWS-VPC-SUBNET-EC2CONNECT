# AWS VPC, Public Subnet & EC2 Instance Setup

## Project Overview
 An Amazon EC2 instance is launched inside the public subnet and connected using EC2 Instance Connect.
This hands-on project is designed to understand how AWS networking components work together to provide internet connectivity to EC2 instances.

## Services Used

* Amazon VPC
* Public Subnet
* Internet Gateway (IGW)
* Route Table
* Amazon EC2
* Security Group
* EC2 Instance Connect
* 
## Architecture
VPC-PUBLIC SUBNET-INTERNET GATEWAY-ROUTE TABLE-EC2 CONNECT
